# jkbFontMaker

A browser-based tool for creating handwritten fonts. Draw each glyph on canvas, preview your typeface in real time, and export it as an OpenType (.otf) file.

**[Try it live](http://play.mattjakob.com/fontmaker)**

## Features

- Draw glyphs with pointer/touch input on an HTML Canvas editor
- Reference font overlay to guide letterform proportions
- Adjustable stroke width and kerning
- Live text preview as you build your character set
- Export to `.otf` via opentype.js
- Import/export projects as JSON for backup and sharing
- All data stored locally in the browser (localStorage)

## Tech Stack

Vanilla JS, HTML Canvas 2D, Vite, opentype.js
